## 5. Product-Market Fit (PMF) Assessment

### 5.1 PMF Signals

**Positive Signals:**
- [ ] Customers willing to pay before seeing the product
- [ ] Customers proactively recommend to other companies
- [ ] NPS > 30
- [ ] Customer usage frequency and depth growing
- [ ] Customers expanding usage scope and user count

**Negative Signals:**
- [ ] High churn rate
- [ ] Many demos but few conversions
- [ ] Customers only using basic features
- [ ] Heavy customization required to use
- [ ] Sales cycles getting longer

### 5.2 Sean Ellis Test

> "If you could no longer use [Product Name], how would you feel?"
> - Very disappointed → Strong PMF signal
> - Somewhat disappointed → Moderate PMF signal
> - Indifferent → Weak PMF signal
